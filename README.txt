Instructions
============
- Install vagrant
- `vagrant up`
- Add these two entries to your hosts file
```
10.10.10.159    shop.vm www.shop.vm
10.10.10.159    php.vm www.php.vm
```
- Custom code should be written in the code folder and can be accessed here [http://php.vm](http://php.vm)
- Shop code will be written in the shop folder and can be accessed here [http://shop.vm](http://shop.vm)
